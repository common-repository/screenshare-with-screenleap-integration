== Description ==
The plugin allows screen-share meetings in wordpress by integrating ScreenLeap.

= Features =

* Create screen-share meeting using any of the three types of screenleap meetings [ Meet Now - Schedule Meeting - API ].
* Easy customizable meeting custom post type details.
* Built-in countdown timer until the meeting date.
* Allows access meetings for logged-in users only.

[Pro Version](https://grandplugins.com/product/screenshare-with-screenleap-integration-pro/)


* Add reminder subscription form for user to remind them before meeting start.
* Make the meeting purchasable by linking with a WooCommerce Product.
* Full Access to the API configurations.
* Display Logged Viewers of the screen-share

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fill the API requirements in 'Screeleap Meetings'/'Settings'
